## Setup
⚠️ These instructions assume that you have yarn. If you don't, make sure to install it: `npm i -g yarn`

```bash
git clone https://github.com/amilajack/erb-node-fibers-example.git
cd erb-node-fibers-example && yarn
cd app && yarn
cd ..
yarn dev
```
